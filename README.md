# Abhishek Yadav — Portfolio

A modern, dark-themed portfolio website built as a single HTML file.

## 🚀 Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
cd portfolio-folder
vercel
```

### Option 2: Vercel Dashboard
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Drag & drop this folder (or connect GitHub)
4. Click Deploy!

### Option 3: GitHub → Vercel
1. Push this folder to a GitHub repo
2. Import on Vercel dashboard
3. Auto-deploys on every push

## Files
- `index.html` — The complete portfolio (all CSS/JS inline)
- `vercel.json` — Vercel deployment configuration
- `README.md` — This file

No build step needed — it's pure HTML/CSS/JS!
